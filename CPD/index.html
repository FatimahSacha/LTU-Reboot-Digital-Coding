Today I leart about the structure of HTML and the four main tags that make up the HTML structure. These are !doctype html, head, title and body all enclosed in left-angled bracket and a right angled bracket.
I also learnt that a HTML element is defined by a start tag, some content, and an end tag. Attributes can also be found in elements and this gives more information to the browser about that element. Introduction to CSS

What have I learnt?

Today I learnt that CSS is the language that is used to style how the elements in html will be displayed and how it can control multiple web pages all at once. The CSS syntax consists of a selecore and a declaration block. An example of this is h1 { color:red;}. Within CSS you can have multiple declarations for one selector.

When creating and designing a web page the design and layout is important. In CSS the box model is used to represent this. The box model allows us to add a border around elements, and to define space between elements.

How did I use this in practice?

In the Google clone page I started to style the header. I changed the colour to grey and the font size to 18px.

To get a better understanding of CSS I used Freecodecamp and W3 Schools to understand how CSS controls the styling of the HTML elements.

Introduction to Javascript

What have I learnt?

Today I learnt that Javascript is a client-side scripting language thats works on browsers and soemtimes on the server side aswell. Javascript is the third layer used in web development for functionality after HTML and CSS. I also learnt that when a web page is loaded, the browser creates a Document Object Model (DOM) of the page.

Javascript is a series of instructions that a computer follows. This can be through psuedocode or varaiables

How did I use this in practice?
