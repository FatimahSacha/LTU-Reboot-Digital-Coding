Introduction to CSS

What have I learnt?

Today I learnt that CSS is the language that is used to style how the elements in html will be displayed and how it can control multiple web pages all at once. The CSS syntax consists of a selecore and a declaration block. An example of this is h1 { color:red;}. Within CSS you can have multiple declarations for one selector.

When creating and designing a web page the design and layout is important. In CSS the box model is used to represent this. The box model allows us to add a border around elements, and to define space between elements.

How did I use this in practice?

In the Google clone page I started to style the header. I changed the colour to grey and the font size to 18px.
